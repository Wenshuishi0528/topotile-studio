incomplete, outdated, simplified, or geometrically incorrect.

数据完整性取决于 OpenStreetMap 的标注质量、公共 API 可用性、选择范围大小和本地生成参数。

`.stl` file is watertight, manifold, printable, or compatible with every slicer
