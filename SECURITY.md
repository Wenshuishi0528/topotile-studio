# Security Policy / 安全说明

TopoTile Studio is designed to run locally on your own computer.

TopoTile Studio 设计为在你自己的电脑本地运行。

## Supported Use / 推荐使用方式

- Run it on `127.0.0.1` / `localhost`.
- Do not expose the development server directly to the public internet.
- Do not upload private DEM files, route files, or project files to services
  you do not trust.
- Generated jobs and caches are stored locally under the project directory.

## Reporting Security Issues / 报告安全问题

If you find a security issue, please open a GitHub issue with a minimal
description and avoid posting sensitive private data.

如果你发现安全问题，请在 GitHub issue 中提交最小化描述，避免公开敏感私人数据。
